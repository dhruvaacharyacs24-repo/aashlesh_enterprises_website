
 # Aashlesh Enterprises

Aashlesh Enterprises, based in Gangolli, is a trusted business dealing in plywood dealership, plastic tarpaulins, and a wide range of interior products, along with other related building and home improvement essentials.

## About Us

Located in Gangolli, Aashlesh Enterprises has established itself as a reliable name for customers looking for quality plywood, durable plastic tarpaulins, and interior solutions to complete their homes and workspaces.

## Our Offerings

- **Plywood Dealership** — A wide range of plywood boards for construction, furniture, and interior applications.
- **Plastic Tarpaulins** — Durable and weather-resistant tarpaulins for various uses.
- **Interior Items** — Products and materials to enhance and complete interior spaces.

## Tech Stack

This website is built with a modern, high-performance web stack:

- **Framework:** [TanStack Start](https://tanstack.com/start) with [TanStack Router](https://tanstack.com/router) and [TanStack Query](https://tanstack.com/query)
- **UI Library:** React 19
- **Language:** TypeScript
- **Styling:** Tailwind CSS 4
- **Component Primitives:** Radix UI (accordion, dialog, dropdown, tabs, tooltip, and more), assembled in a shadcn/ui-style component system (class-variance-authority, cmdk, vaul, tailwind-merge)
- **Forms & Validation:** React Hook Form with Zod
- **Backend / Database:** Supabase
- **Build Tool:** Vite 8, with Nitro as the server runtime
- **Other Notable Libraries:** Recharts (charts), Embla Carousel, date-fns, Sonner (toasts), lucide-react (icons)
- **Code Quality:** ESLint + Prettier, TypeScript ESLint
- **Scaffolded with:** Lovable (via `@lovable.dev/vite-tanstack-config`)
